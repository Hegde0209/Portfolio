Egen Solutions Inc.

August 2015 - October 2015
Project: Customer Portfolios Marketing Workbench
Role: Web Developer
Description:

The Customer Portfolios Marketing Workbench (Workbench) is a web application helping marketers plan, develop, and execute multi-channel closed-loop marketing programs. The goal of the Workbench project was to allow marketers to create standard reusable list queries against customer data stored in the CP Data Marts without writing SQL queries. The functionality is intended to be used by thousands of non-technical customers concurrently.

Responsibilities:
  • Implementation of User Interface using HTML5, SASS, AngularJS with Ionic Framework and used Gulp as a task runner with Jasmine as a framework for Unit Test cases.
  • Implemented project using Agile SCRUM methodology. Involved in daily stand up meetings, sprint showcase and sprint retrospective.
  • Worked on performance issues of the application using CSS Sprites, Minification and Concatenation of files
  • Worked on Customized Directives, Controllers, Implementation of Routing and Pagination using features of Angular libraries
  • Backend was supported by Java Spring MVC and Elastic search and also have contributions in developing APIs in Spring MVC
  • Designed and implemented MVC architecture using Spring Framework and functionality to fetch data from the database.
  • Involved in writing application level code to interact with REST APIs, Web Services using AJAX, JSON and XML
  • Experience in making Responsive web pages using Twitter Bootstrap and media queries.
  • Implemented Log4J for Logging Errors, debugging and tracking.
Technologies Used:Java, Spring MVC Framework, HTML5, LESS, AngularJS, JavaScript, REST APIs, AJAX, JSON, XML, Gulp, Karma - Jasmine, Log4J, Apache Tomcat, Git
